# Input
The input of this application is the user’s interaction. The user selects a movie genre from a dropdown menu and clicks on a movie item. The movie data (title, genre, and release year) is predefined in the application.

# Process
The application uses React functional components and the useState hook to manage the selected genre. When the user selects a genre, the application filters the movie list based on the selected value. The filtering is done using JavaScript array methods. When the user clicks on a movie, an alert is triggered to display the movie title.

# Output
The output is a dynamic movie list displayed on the screen. The list updates automatically when the selected genre changes. When a movie is clicked, an alert box appears showing the movie title.
